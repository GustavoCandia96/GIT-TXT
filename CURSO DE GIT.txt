CURSO DE GIT
El dia 21/09/2022 he rotomado el curso de git en el canal de pildorasInformaticas en el primer video aprendi 
a como instalar Git.
Git es una herramienta poderosa que todo informatico debe de conocer, la funcion de git es dar seguimiento a los proyectos 
que se va realizando y no perderlos realizando copias de seguridad en el repositorio local y cuando sea el momento de alzarlo 
a la nube hacerlo correctamente mediante comandos.
El primer comando aprendido en el curso es el comando git init
Este comando se debe de realizar en donde se encuentra el proyecto clic derecho y ejecutar Git Bash Here
Se hace de esta manera para que Git pueda detectar la raiz de la ubicacion del proyecto.
El segundo comando es git status -s, este comando se utiliza para ver todos los archivos que se encuentra dentro de la carpeta raiz
El tercer comando es git add que se utiliza para agregar dentro del repositorio local y hacer seguimiento git add mas el nombre del archivo
git commit -m es el comando que nos permitira dar nombre al repositorio Ejemplo git commit -m "Ejemplo del nombre"
git log --oneline es un comando que nos ayudara a ver todas las ramificaciones que hemos estado realizando en el proyecto
